# Pre-submission repository checklist

Before linking the repository in the manuscript:

- [ ] README explains the paper, method, data policy, and setup.
- [ ] Raw datasets and checkpoints are not committed.
- [ ] Large cached logits are not committed unless explicitly allowed.
- [ ] Final notebooks have clean names and cleared heavy outputs.
- [ ] SASC/Frozen V3 config is fixed in `configs/frozen_v3_sasc.json`.
- [ ] Core scripts run from the repository root.
- [ ] Results and figures included are small and directly used in the paper.
- [ ] `CITATION.cff` has the final author list and paper details.
- [ ] The repository URL in the manuscript is correct.
